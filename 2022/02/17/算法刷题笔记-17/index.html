最大正方形

最大正方形

解题思路

利用动态规划，设dp[i][j]为以matrix[i][j]为右下角的只包含1的正方形的边长：

  • 初始化dp数组：

    • 若matrix[i][0] == '1'，dp[i][0] = 1，否则为0
    • 若matrix[0][j] == '1'，dp[0][j] = 1，否则为0

  • 更新dp数组：

    • 若matrix[i][j] == '1'，dp[i][0] = 1 + min(dp[i-1][j-1], dp[i-1][j], dp[i][j-1])
    • 若matrix[i][j] == '0'，dp[0][j] = 0

利用遍历max记录最长的边长即可，最大面积 = 边长 * 边长
